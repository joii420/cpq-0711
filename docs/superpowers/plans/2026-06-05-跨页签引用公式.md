# 跨页签引用公式（cross_tab_ref token）实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让 B 页签的列公式（FORMULA 字段）能按"A.列 = B.列（多列 AND）"匹配同卡片内 A 页签的行，取值（需求1）或聚合 SUM/AVG/COUNT/MAX/MIN（需求2），结果可嵌入任意列公式。

**Architecture:** 新增公式 token 类型 `cross_tab_ref`，与既有 `component_subtotal`/`previous_row_subtotal` 同族。在前端 `formulaEngine.ts` 与后端 `FormulaCalculator.java` 两个 token 引擎里**对等**实现求值；在前端 `QuotationStep2#computeAllFormulas`（及 `ReadonlyProductCard`/`useCardSnapshots`）与后端 `CardSnapshotService` 两处编排里，按组件级依赖**拓扑排序**并维护"同卡片已算行存储"。保存时在 `ComponentService#validateFormulas` 校验。

**Tech Stack:** Java 17 + Quarkus（后端 token 引擎、JUnit5）；React + TypeScript（前端 token 引擎、Vitest、Playwright E2E）；Ant Design Drawer（配置面板）。

**设计依据：** `docs/superpowers/specs/2026-06-05-跨页签引用公式-design.md`（路线甲）。

## 数据结构契约（全计划共用，先读）

### token JSON 形态

```jsonc
{
  "type": "cross_tab_ref",
  "source": "<A 组件标识>",      // 运行时匹配键：优先 componentId，退 componentCode
  "sourceLabel": "投料",          // 回显用
  "target": "<A 目标列字段名>",    // agg=COUNT 时允许空串
  "match": [ { "a": "A匹配列名", "b": "B匹配列名" } ],   // 多项 = AND
  "agg": "NONE"                   // NONE | SUM | AVG | COUNT | MAX | MIN
}
```

### "已算行存储"形态（两端一致）

- 类型：`组件标识 → 该组件所有数据行`，每行 = `字段名 → 已算值（原始类型，文本或数字）`。
- 不含 SUBTOTAL 组件、不含小计行。
- 后端复用 `CardSnapshotService` 既有 `resolvedRows`（"按字段名标量行"），前端复用 `computeAllFormulas` 每行算出的 `fieldValues` + 原始行合并值。

### 求值语义（两端一致）

1. 取 `crossTabRows[token.source]`；缺失（组件不存在/未算）→ 视为 0 行。
2. 逐行匹配：对 `match` 每项，比较 `Arow[a]` 与 `Brow[b]`（B 当前行原始值）；任一方 null/空串 → 该行不匹配；全部相等才命中。
3. 出值：
   - `NONE`：命中 0 → `0`；命中 1 → `Arow[target]`；命中 >1 → **错误**（前端 throw / 后端返回错误哨兵 → 整公式按 0/—）。
   - `SUM/AVG/MAX/MIN`：对命中行 `target` 取数字（非数字 → **错误**）后聚合；空集 → `0`。
   - `COUNT`：命中行数；空集 → `0`。

```java
    /** 多匹配/非数字聚合错误哨兵。 */
    private static final class FormulaErrorMarker {}
    private static final FormulaErrorMarker ERR = new FormulaErrorMarker();

    /** cross_tab_ref 求值。返回 Number / String（NONE 文本）/ ERR。 */
    Object evalCrossTab(JsonNode token, RowContext ctx) {
        String source = token.path("source").asText("");
        String target = token.path("target").asText("");
        String agg = token.path("agg").asText("NONE").toUpperCase();
        List<Map<String, Object>> rows = ctx.crossTabRows.getOrDefault(source, List.of());

        List<Map<String, Object>> hits = new ArrayList<>();
        for (Map<String, Object> arow : rows) {
            boolean ok = true;
            for (JsonNode pair : token.path("match")) {
                Object av = arow.get(pair.path("a").asText(""));
                Object bv = ctx.currentRowRaw.get(pair.path("b").asText(""));
                if (isBlank(av) || isBlank(bv) || !valEquals(av, bv)) { ok = false; break; }
            }
            if (ok) hits.add(arow);
        }

        if ("COUNT".equals(agg)) return java.math.BigDecimal.valueOf(hits.size());
        if ("NONE".equals(agg)) {
            if (hits.isEmpty()) return java.math.BigDecimal.ZERO;
            if (hits.size() > 1) return ERR;
            return hits.get(0).get(target);
        }
        // SUM/AVG/MAX/MIN
        if (hits.isEmpty()) return java.math.BigDecimal.ZERO;
        List<Double> nums = new ArrayList<>(hits.size());
        for (Map<String, Object> h : hits) {
            Double n = toNumber(h.get(target));
            if (n == null) return ERR; // 非数字聚合 → 错误
            nums.add(n);
        }
        double r;
        switch (agg) {
            case "SUM": r = nums.stream().mapToDouble(Double::doubleValue).sum(); break;
            case "AVG": r = nums.stream().mapToDouble(Double::doubleValue).average().orElse(0); break;
            case "MAX": r = nums.stream().mapToDouble(Double::doubleValue).max().orElse(0); break;
            case "MIN": r = nums.stream().mapToDouble(Double::doubleValue).min().orElse(0); break;
            default: return ERR;
        }
        return java.math.BigDecimal.valueOf(r);
    }

    private static boolean isBlank(Object o) {
        return o == null || (o instanceof String s && s.isBlank());
    }

    /** 匹配键相等比较：数字按数值，否则按 trim 文本。 */
    private static boolean valEquals(Object a, Object b) {
        Double na = toNumber(a), nb = toNumber(b);
        if (na != null && nb != null) return na.doubleValue() == nb.doubleValue();
        return String.valueOf(a).trim().equals(String.valueOf(b).trim());
    }
```

> `toNumber(Object)` 已存在于 FormulaCalculator（path/gvar 复用）。若签名为 private static 可直接用；确认其能处理 BigDecimal/Number/String。

```java
package com.cpq.quotation.service;

import com.cpq.common.exception.BusinessException;

import java.util.*;

/** 组件级 cross_tab_ref 依赖拓扑排序（Kahn）+ 环检测。入度=本组件依赖数；稳定保留输入相对序。 */
public final class CrossTabComponentOrder {
    private CrossTabComponentOrder() {}

    /**
     * @param components 组件标识列表（按出现顺序）
     * @param deps       组件标识 → 它依赖的源组件标识集合（仅含 components 内的）
     * @return 拓扑序；存在环抛 BusinessException(400)
     */
    public static List<String> topoOrder(List<String> components, Map<String, Set<String>> deps) {
        Map<String, Integer> indeg = new LinkedHashMap<>();
        for (String c : components) indeg.put(c, 0);
        for (String c : components) {
            for (String d : deps.getOrDefault(c, Set.of())) {
                if (indeg.containsKey(d)) indeg.merge(c, 1, Integer::sum);
            }
        }
        Deque<String> q = new ArrayDeque<>();
        for (String c : components) if (indeg.get(c) == 0) q.add(c); // 保留输入序
        List<String> order = new ArrayList<>();
        while (!q.isEmpty()) {
            String c = q.poll();
            order.add(c);
            for (String other : components) {
                if (deps.getOrDefault(other, Set.of()).contains(c)) {
                    indeg.put(other, indeg.get(other) - 1);
                    if (indeg.get(other) == 0) q.add(other);
                }
            }
        }
        if (order.size() != components.size()) {
            Set<String> cyc = new LinkedHashSet<>(components);
            cyc.removeAll(order);
            throw new BusinessException(400, "页签公式存在循环引用: " + cyc);
        }
        return order;
    }
}
```

    /** 扫描一个组件 formulas 节点（[{expression:[token...]}]），收集所有 cross_tab_ref 的 source。 */
    public static java.util.Set<String> extractSourceRefs(
            com.fasterxml.jackson.databind.JsonNode formulas) {
        java.util.Set<String> refs = new java.util.LinkedHashSet<>();
        if (formulas == null || !formulas.isArray()) return refs;
        for (var f : formulas) {
            var expr = f.path("expression");
            if (!expr.isArray()) continue;
            for (var tk : expr) {
                if ("cross_tab_ref".equals(tk.path("type").asText(""))) {
                    String s = tk.path("source").asText("");
                    if (!s.isBlank()) refs.add(s);
                }
            }
        }
        return refs;
    }
```

import type { ComponentFormula } from './types'; // 若类型路径不同按实际改

export function extractSourceRefs(formulas?: Array<{ expression?: any[] }>): string[] {
  const refs = new Set<string>();
  for (const f of formulas ?? []) {
    for (const t of f.expression ?? []) {
      if (t?.type === 'cross_tab_ref' && t.source) refs.add(t.source);
    }
  }
  return [...refs];
}

/** Kahn 拓扑序；保留输入相对序；有环 throw。deps: 组件标识→依赖的源组件标识[]。 */
export function topoOrderComponents(ids: string[], deps: Record<string, string[]>): string[] {
  const indeg: Record<string, number> = {};
  ids.forEach((c) => (indeg[c] = 0));
  ids.forEach((c) => (deps[c] ?? []).forEach((d) => { if (d in indeg) indeg[c]++; }));
  const q = ids.filter((c) => indeg[c] === 0);
  const order: string[] = [];
  while (q.length) {
    const c = q.shift()!;
    order.push(c);
    ids.forEach((o) => {
      if ((deps[o] ?? []).includes(c)) { indeg[o]--; if (indeg[o] === 0) q.push(o); }
    });
  }
  if (order.length !== ids.length) throw new Error('cross_tab_ref 组件循环引用');
  return order;
}
```
