📍 设计说明：
• 三类 BOM 同页：业务在编辑/复核 BOM 时常需要在「组成件 → 元素 → 来料」三层之间穿梭对照，独立成 3 个菜单需要反复跳转。聚合在一页通过 Tab 切换，同时保留切换时的筛选/搜索条件。
• 列结构差异化：每个 Tab 的字段不同（组成件含"损耗率"，元素 BOM 含"工艺路径"，来料 BOM 含"供应商/采购单价"），各 Tab 内部独立列定义，互不影响。
• 路由：/bom?type=component | element | material，3 个 Tab 共享主搜索框（搜父料号时自动同步过滤）。
