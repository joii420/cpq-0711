📌 弱关联机制说明（v0.4 优化 2 核心设计）：
  • 选配实例可独立存在（不必关联报价单），如内部库 / 客户尚未确定时
  • 选配实例可绑定到任意已有报价单（点击 [绑定到报价单] 按钮选）
  • 选配实例可从这里直接新建报价单（自动绑定，但二者数据无前后关系）
  • 选配实例可解绑，不影响已存在的报价单
  • 删除选配实例 ≠ 删除报价单（双向独立，ON DELETE SET NULL）
  • 删除报价单 ≠ 删除选配实例（仅清空 linked_quotation_id）
