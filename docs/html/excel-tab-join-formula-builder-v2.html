求值规则（按顶层 + - 拆"加减项"，逐项判定）： 项内有"裸明细字段"(未被函数圈住)→整项按对齐行逐行算再求和；明细全在函数内/整项纯标量→只算一次。
例：[投料.金额]*[加工.工时]=Σ(每行 金额×工时)； AVG([投料.工时])=均值(一次)； AVG([投料.工时])*[投料.数量]=均值×Σ数量； MAX([投料.金额])+[回料(总计)]=最大金额+回料总计(一次)。
