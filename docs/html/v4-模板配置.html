📊 版本比对 取代菜单

选 2 行（同模板的两个版本 / 不同模板）→ 点按钮 → 右侧抽屉以左右对比展示字段、公式、组件、绑定差异；红 = 删除，绿 = 新增，黄 = 修改。

enabledWhen: selectedRows.length === 2

🔗 绑定产品 取代菜单

选 1 行 → 抽屉展示当前模板已绑定的产品 + 待绑定候选；支持批量勾选、按分类/标签筛选、保存即生效（写入 template_product 关联表）。

enabledWhen: selectedRows.length === 1

🏷️ 业务标签 取代菜单

无需选行（全局字典）→ 抽屉以表格管理标签 key / 中文标签 / 颜色 / 排序 / 启用；保存后立即在本页"业务标签"列显示。

enabledWhen: true（始终可用）
