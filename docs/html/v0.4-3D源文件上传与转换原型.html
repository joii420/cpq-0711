📖 从 UG NX 导出 STEP 文件 — 操作指南

🎯 操作目标

在 UG NX 内打开你的产品 .prt 设计文件 → 通过 File → Export → STEP 导出标准 STEP 格式（推荐 AP214 或 AP242 版本） → 同时保存原 .prt + 新生成 .stp → 上传到 CPQ。

📋 详细步骤

Step 1. 打开 UG NX，确保已加载你要导出的 .prt 装配体或单件
📷 UG NX 主界面截图占位
Step 2. 主菜单选 File → Export → STEP203 / STEP214 / STEP242
💡 推荐用 STEP214（兼容性最好），新版 UG 用 STEP242（含 PMI 信息）
Step 3. 弹出导出对话框，设置：
  • 勾选 Save Solid Models / Save Surface Models
  • 勾选 Single File Mode（单文件输出，便于上传）
  • 单位选 Millimeters（毫米，与 CPQ 标准一致）
  • 路径建议：{partNo}-v{N}.stp
Step 4. 点 OK 等待导出（小件 5s / 大装配 60s+）→ 生成 .stp 文件
Step 5. 同时保留原 .prt 文件（CPQ 永久保存用于后续设计变更追溯）

⚠️ 常见问题

Q: 导出后 STEP 文件太大（>10MB）？
A: ① 在 UG 内 Simplify Geometry 删除冗余特征 → 再导出；② 装配体太大可拆分上传
Q: 我用的是 Solidworks / CATIA / Pro-E?
A: 同样可导出 .stp，但 CPQ 暂不保存原 .sldprt / .CATPart / .prt.* 文件，只保存导出的 .stp + 生成的 .glb。详见 docs/CAD导出GLB操作手册.md
Q: 导出 STEP 后还要不要先在 Blender 转一次？
A: 不需要！CPQ 后端自动用 FreeCAD + Blender 转换。**不要**在 UG 外手工转 GLB（会丢精度，且失去重新转换的能力）
💡 完整 CAD 软件导出对照参考：CAD导出GLB操作手册.md
