## 7. 费用表 `fee_config`

> 聚焦商务/物流类费用配置，涵盖利润率、税率、运费、清关费、保险费、银行手续费等，支持按料号、客户、地区维度配置。

| 字段 | 类型 | 必填 | 含义 |
|------|------|:----:|------|
| `system_type` | VARCHAR(10) | ✅ | 适用系统：QUOTE / PRICING |
| `biz_type` | VARCHAR(30) | ✅ | 费用大类：PROFIT 利润率 / TAX 税率 / FREIGHT 运费 / CUSTOMS 清关费 / INSURANCE 保险费 / BANK 银行手续费 / OTHER 其他/管理费 |
| `fee_no` | VARCHAR(30) | ✅ | 费用项编号 |
| `fee_name` | VARCHAR(100) | ✅ | 费用项名称（如：销售利润率、增值税率、海运费、空运费、清关代理费） |
| `material_no` | VARCHAR(20) | | 料号（按料号配置时填，全局费率可为空） |
| `customer_no` | VARCHAR(20) | | 客户编号（按客户配置时填） |
| `region` | VARCHAR(50) | | 地区/国家（按地区配置运费/清关时填） |
| `charge_basis` | VARCHAR(20) | | 计费基准：RATE 按比例 / FIXED 固定金额 / PER_UNIT 单位金额 / PER_KG 按重量 |
| `value` | DECIMAL(18,6) | | 固定值/单位金额 |
| `ratio` | DECIMAL(10,4) | | 比例（%）（利润率、税率等） |
| `currency` | VARCHAR(10) | | 货币 |
| `unit` | VARCHAR(20) | | 计价单位 |
| `effective_date` | DATE | | 生效日期 |
| `expire_date` | DATE | | 失效日期 |
| `pricing_version_no` | VARCHAR(20) | | 核价版本编号 |

## 9. 电镀方案表 `plating_scheme`

> 定义电镀方案的元素配置，包含电镀方式、表面积、镀层厚度及理论元素用量等参数。

| 字段 | 类型 | 必填 | 含义 |
|------|------|:----:|------|
| `scheme_no` | VARCHAR(20) | ✅ | 电镀方案编号 |

| `seq_no` | INT | ✅ | 项次 |

| `plating_method` | VARCHAR(30) | ✅ | 电镀方式：电镀 / 化学镀 / 蒸镀 / 浸镀 / 刷镀 / 整体镀 / 局部镀 |

| `plating_thickness` | DECIMAL(18,6) | ✅ | 镀层厚度（μm） |
| `plating_requirement` | VARCHAR(200) | | 电镀要求/规格描述 |
| `density` | DECIMAL(18,6) | | 密度（g/cm³） |
| `element_usage` | DECIMAL(18,6) | ✅ | 元素用量（单件产品消耗，理论值 = plating_area × plating_thickness × density） |
| `element_usage_unit` | VARCHAR(20) | | 元素用量单位（g / mg / kg / pcs） |
| `effective_date` | DATE | | 生效日期 |
