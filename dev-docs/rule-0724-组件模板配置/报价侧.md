## 契约要点

- **客户口径**：`:customerCode` 按客户 `customer_no` 收窄；dry-run 时 NULL 兜底。
- **料号键**：`material_no`（销售料号）`AS hf_part_no`。
- **版本**：无——提交即冻结（frozen 逐字节隔离），不可切版本。
- **渲染服务**：`ConfigureSnapshotService`；两趟路由——Pass2 树渲染（仅 tabType=BOM，注 `:total_material_no`）、Pass1 平铺展开（其余页签，注 `:customerCode`+hf_part_no）。**数据页签禁用 `:total_material_no`**（Pass1 不注入 → 静默 0 行）。

## §V6 表映射（字段来源权威表）

- 产品（主件）：`material_customer_map` + `material_master`
- 材料成本（材质元素）：`element_bom_item`（材质经本行 `material_part_no`→`material_recipe`；一号多材质铁律）
  - **元素单价接客户价格策略**（task-0722，来源 `报价模板生成规则.md §5.2.1`）：`LEFT JOIN f_customer_element_price(:customerCode, :priceBaseDate) cep ON cep.element_code = ebi.component_no`，输出 `cep.unit_price AS 单价, cep.currency AS 货币`；required_variables `["customerCode","priceBaseDate"]`；硬约束：LEFT JOIN / 键=元素符号 component_no / 禁 COALESCE 兜底 / 不绑计价单位(那是 BOM 发料单位 issue_unit) / 别名逐字一致。
- 加工费/自制加工费（零件）：`unit_price` PROCESS（要素名=`process_master` 工序名；料号=`code` 零件料号）
- 电镀费：`unit_price` PLATING（行转列 PIVOT，或逐元素）
- 外购件：`unit_price` COMPONENT_OTHER（费用）或 `material_bom_item` OUTSOURCED（组成件）
- 其他费用：`unit_price` FINISHED_MATERIAL_OTHER + INCOMING_MATERIAL_OTHER
