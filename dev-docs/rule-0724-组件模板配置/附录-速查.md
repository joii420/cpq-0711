## A. 报价 vs 核价 6 维度对照（一屏看全）

| 维度 | 报价侧 | 核价侧 |
|---|---|---|
| 客户口径 | `:customerCode` 按客户收窄 | 无 customerCode，`_GLOBAL_` 全局成本 |
| 版本 | 无（提交即冻结） | `:versionFilter` 宏，每 sheet 独立可切（仅 PENDING） |
| 树 | 可选（tabType=BOM），数据页签平铺 | 核心主轴，每组件 spine 全节点补空行 |
| 料号键 | `material_no`（销售料号） | 销售料号 JOIN `material_master`→生产料号 |
| 定价口径 | `unit_price` 按 `customer_no` | `_GLOBAL_` |
| 渲染服务 | `ConfigureSnapshotService`（Pass1平铺+Pass2树） | `CardSnapshotService`+`BomTreeRenderService`（spine） |

## B. 常见坑速查

| 坑 | 症状 | 修 |
|---|---|---|
| 禁表（AP-53） | dry-run 拒 / 渲染断链 | FROM V6 表，`$view` 引用 |
| pending 改写（版本化表 LEFT JOIN 含 is_current） | `column index out of range` 静默 0 行，dry-run 照过 | 改标量子查询 |
| 视图 ORDER BY 失效（DRAFT） | 多行页签倒序 | 用组件 `sort_field` |
| 料号列绑成品料号 | 树把成品误判零件 | 零件绑 `unit_price.code` 非 `finished_material_no` |
| 中文标识符 | 公式/路径解析失败 | ASCII 别名或 Java 预解析 |
| 缓存 key 缺 componentId | 同名视图跨组件串号 | 缓存 key 含 componentId+fieldsHash+total_material_no |
| 迁移号冲突（并发会话） | 我的迁移被同号跳过，8081 挂 500 | 建迁移前 `find` 全 worktree + 查 history max |
| 一号多材质撞键 | editRows 塌缩（末值×行数） | 材质字段进 rowKeyFields |
| 数据页签用 `:total_material_no` | 报价数据页签静默 0 行 | 数据页签用 `:customerCode`+hf_part_no 平铺契约 |

## C. V6 现役表映射速查

| 表 | 用途 | 关键列 |
|---|---|---|
| `material_master` | 成品/物料主档 | material_no, material_name, standard_unit, material_recipe_id |
| `material_customer_map` | 客户料号映射（占号唯一） | material_no, customer_no, customer_material_name, exchange_rate |
| `material_bom_item` | BOM（父子边） | material_no(父), component_no(子), characteristic(RECIPE/ASSEMBLY/OUTSOURCED) |
| `element_bom_item` | 元素明细 | material_no(成品), material_part_no(材质料号), component_no(元素), customer_no |
| `material_recipe` | 材质配方 | code, name, symbol |
| `unit_price` | 单价（工序/元素/电镀/费用） | code(零件料号), finished_material_no(成品), price_type, operation_no, customer_no |
| `process_master` | 工序主表 | process_no, process_name |
| `capacity` | 产能/组装加工费 | material_no, process_name, fixed_cost（无 customer_no） |
| `plating_fee`/`plating_scheme` | 电镀费用列式 | plating_process_fee, plating_material_fee / scheme_no |

> ⚠️ 废弃禁用：`mat_*` / `plating_plan` / `element_price*` / `customer_tax` / `costing_part_*`。
