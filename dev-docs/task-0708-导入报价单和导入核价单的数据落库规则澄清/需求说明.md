## 2. 需求背景与目标 ⭐

- **要解决什么问题 / 痛点**：

  由于增加了销售料号的概念,销售料号(宏丰料号 excel 列改名为销售料号)作为本系统的主料号,本周对导入报价单数据入库和核价单数据入库的规则进行了修改,与我当前的修改方案有冲突, 增加了sale_material_no字段.现在相反,原先的material_no就作为销售料号,现在应该增加的是product_no字段用来存储生产料号

  现在要求规则统一,对比7月5号之前的导入规则,将原定的生产料号语义字段统一改为销售料号的语义字段进行落库,落在原先的字段上,同时新增生产料号字段

  excel中列名澄清 :

  核价: 物料料号-> 材质料号

- **本次要达成的目标**：
  
1.将功能实现回退到之前的样子,然后按现在的要求进行修改,
  
  2.纠正现在的文档 docs/table目录下的 报价系统Excel导入落库方案.md 和 核价系统Excel导入落库方案.md 文档,补充销售料号和生产料号的内容.
  
  3.报价导入能正确数据落库 : 测试excel文件:  docs/table/报价测试数据/报价系统功能基础数据功能结构所需字段V3.xlsx
  
  4.核价导入能正确数据落库 : 测试excel文件:  docs/table/核价测试数据/核价系统功能基础数据功能结构所需字段（6.0版） .xlsx
  
- **不做什么（明确边界）**：
